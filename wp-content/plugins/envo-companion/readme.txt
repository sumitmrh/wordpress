Envo Companion is a companion plugin for Webenvo themes.

== Description ==

This is companion plugin is created for Envo themes. 
The plugin add custom frontpage sections for Envo themes and it will allow you full customizer settings for the theme front page sections. 
The plugin options are user-friendly and you can use this plugin for all envo child themes also.
